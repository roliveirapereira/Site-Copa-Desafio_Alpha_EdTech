# 🇧🇷 Seleção Brasileira - Tributo à Amarelinha

Este projeto foi desenvolvido como parte da **Questão 10 do módulo de CSS**, submetido para o **Mini Desafio T8 - Turma Toyota** da **Alpha EdTech**. 

A proposta foi reaproveitar a estrutura de uma landing page/portfólio e transformá-la de forma criativa em uma página de tributo à Seleção Brasileira de Futebol, celebrando nossos 5 títulos mundiais e a história do futebol arte. Tudo isso respeitando as regras do desafio: utilizando estritamente os conhecimentos de HTML e CSS vistos em aula.

---

## 🎯 Objetivo do Desafio

* Demonstrar o domínio prático de **HTML5** e **CSS3**.
* Aplicar conceitos de organização de código e estruturação semântica.
* Exercitar a criatividade alterando o contexto visual de um projeto base.
* Praticar *soft skills* através da gravação de um "pitch" de apresentação do projeto.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização, layout (Flexbox), variáveis de cor, transições e animações de background baseadas em rolagem (scroll timeline).
* **Git/GitHub:** Versionamento e hospedagem do repositório.

---

## ✨ Funcionalidades e Destaques

* Layout totalmente responsivo, adaptando-se a desktops, tablets e smartphones.
* Efeitos de `hover` nos cards e transições suaves nas imagens.
* Uso de variáveis CSS (Custom Properties) para facilitar a manutenção das cores do tema (Verde e Amarelo).
* Animações de zoom e escurecimento do background na seção principal.

---

## 🚀 Como executar o projeto

Como o projeto utiliza apenas tecnologias nativas da web (HTML e CSS), não é necessária nenhuma instalação complexa.

1. Faça o clone deste repositório em sua máquina local:
   `git clone [COLE A URL DO SEU REPOSITÓRIO AQUI]`
2. Navegue até a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador web de preferência.
